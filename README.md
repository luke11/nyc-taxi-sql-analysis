My project delves into NYC taxi trips using a publicly available dataset that I found from the NYC Taxi & Limousine Commission. I used SQL to demonstrate revenue trends, demand patterns, and taxi business performance.

Dataset:
NYC Taxi & Limousine Commission
https://www.nyc.gov/site/tlc/about/tlc-trip-record-data.page

Skills Demonstrated:
- SQL joins
- Aggregations
- Window functions
- Data validation
- Metric analysis

Skill Examples:
- Revenue by pickup zone
- Profitable taxi routes
- Trip demands by hour of day
- Payment types and tipping review
- Weekday and weekend demand
